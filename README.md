# 🔮 ラッキー音楽占いアプリ

このアプリは、名前を入力してボタンを押すだけで  
**今日の運勢・ひとことコメント・ラッキー音楽**を占える  
シンプルな Web アプリです。

占い結果はデータベースに保存され、過去の履歴も確認できます。

---

## 🌟 アプリの主な機能

- 名前を入力してワンクリックで占いを実行
- ランダムに以下を表示
  - 今日の運勢（大吉〜大凶）
  - ゆるい一言コメント
  - 今日のラッキー音楽（YouTube動画）
- 占い結果をデータベースに保存
- 過去の占い履歴を時系列で表示（最新10件）

---

## 🖥 使用技術

- Python
- Streamlit（WebアプリUI）
- Supabase（PostgreSQL / データ保存）
- YouTube（動画埋め込み）

## 📜 占い方（概要）

1. 名前を入力
2. 「占ってもらう」ボタンをクリック
3. 運勢・コメント・音楽が表示される
4. 結果は自動的に保存され、履歴に反映される
